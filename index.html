Directives 元件庫

  2. ui-select2
    • 參數
      • ng-options : 同原來的 ngOptions
      • ng-model : 同原來的 ngModel
      • ng-disabled : true -> 停用元件
      • settings (或 data-settings) : select2 初始化設定值 參考文件
